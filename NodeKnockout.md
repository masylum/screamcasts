# Hello dickinabox!

## Deploy instructions

### GitHub — [Team][2], [Repo][3]

~~~sh
git clone git@github.com:nko3/dickinabox.git
~~~

### Nodejitsu — [More details][5], [Handbook][4]

~~~sh
npm install -g jitsu
jitsu login --username nko3-dickinabox --password 88u5YIeiLfgTXIAT
jitsu deploy
~~~
